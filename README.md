# NHK_2023_A_Head_National

## 概要
沖縄2023A、獅子舞頭部のコードです<br>
(※デフォルトブランチ：main)

## 機能
- 首RX軸(横振り)サーボ制御
- 首RY軸(頷き)モータ・ロリコン制御
- 首RZ軸(傾げ)モータ・ロリコン制御
- 顎サーボ制御

## 環境
|          |     |
|   :-:    | --- |
| プログラム | Arm Keil Studio |
| 使用OS | Mbed OS 5.15.7 |
| 使用マイコン | NUCLEO-F446RE |
| 使用言語    | C++ |
| その他   | 弊部独自ライブラリの導入が必須(Privateリポジトリのため、今後依存関係を解消予定) |
